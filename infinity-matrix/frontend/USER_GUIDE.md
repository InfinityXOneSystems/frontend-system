
# Infinity X | User Guide

---

## 1. Getting Started

### 1.1 Creating an Account
1.  Navigate to `/auth`.
2.  Select "Create ID".
3.  Enter your email and a secure passkey.
4.  Select a Starter Plan (Free Trial available).

### 1.2 Your First Login
Upon login, you will be greeted by the **Agent Selection Modal**. Choose your primary interface partner:
*   **Vision Cortex:** Best for analysis and strategy.
*   **Quantum X:** Best for building and coding.
*   **Analyst:** Best for data processing.

---

## 2. Using Vision Cortex
Navigate to the **Vision Cortex** page via the sidebar.

**Features:**
*   **Chat:** Type natural language queries ("Analyze the crypto market").
*   **Image Analysis:** Click the image icon to upload charts or screenshots for analysis.
*   **Modes:** Switch between Manual (you guide it) and Auto (it acts autonomously).

---

## 3. Building Agents
Navigate to **Agent Creator**.

1.  **Identity:** Name your agent and give it a role.
2.  **Personality:** Adjust sliders for Empathy, Confidence, etc.
3.  **Skills:** Select modules (Email, Coding, Data).
4.  **Deploy:** Click "Deploy Agent". It will now appear in your dashboard.

---

## 4. Automation Workflows
Navigate to **Workflow Builder**.

1.  Drag a **Trigger** node (e.g., "New Email").
2.  Drag an **Action** node (e.g., "Agent: Summarize").
3.  Connect the dots to define the flow.
4.  Click **Run Workflow** to test.

---

## 5. Admin Console
*Access restricted to workspace admins.*

*   **Dashboard:** View token usage and costs.
*   **Users:** Invite team members and manage roles.
*   **God Mode:** (Advanced) System-wide resets and simulations.

---

## 6. Troubleshooting
*   **Agent Stuck?** Go to Dashboard > Agents and click "Restart".
*   **API Errors?** Check your API Key settings in the Settings page.
*   **Support:** Contact `help@infinityx.ai` or join our Discord.
