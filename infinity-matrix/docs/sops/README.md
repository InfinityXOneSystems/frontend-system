## SOP Generation

SOPs are automatically generated and updated by the `sop-generator.yml` workflow whenever:
- New workflows are added or modified
- Agent or module changes are committed
- System architecture is updated
- Major features are deployed

## SOP Structure

Each SOP follows this standard structure:
1. **Purpose** - Why this procedure exists
2. **Scope** - What it covers
3. **Responsibilities** - Who/what executes it
4. **Procedure** - Step-by-step instructions
5. **References** - Related SOPs and documentation
6. **Revision History** - Automated update tracking
