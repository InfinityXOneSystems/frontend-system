# Infinity Frontend Consolidated

This repository consolidates the following frontend components:

*   **frontend**: The main frontend application.
*   **admin-dashboard**: The administrative dashboard.
*   **infinity-matrix frontend components**: Frontend components from the infinity-matrix project.

## Structure

Each original repository's content is organized into its own subdirectory:

*   `./frontend`
*   `./admin-dashboard`
*   `./infinity-matrix`

## Getting Started

Refer to the individual subdirectories for specific setup and running instructions.
